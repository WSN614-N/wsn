1.程序简介
本程序是计算力学课程《2.3 系统总体刚度方程》的作业实现，采用模块化设计，基于直接刚度法实现了一维杆单元和二维桁架单元的有限元分析。程序支持自动生成对号矩阵、组装总体刚度矩阵、施加位移边界条件、求解节点位移和约束反力、计算单元应力和轴力，并自动完成总体刚度矩阵的四大性质检查（对称性、奇异性、对角元非负性、稀疏性）。

2.环境要求
Python 3.7 及以上版本
numpy：矩阵运算
matplotlib：桁架结构可视化

3.文件结构
main.py          # 主程序入口，串联全流程
model.py         # 有限元模型数据类，存储所有参数和结果
element.py       # 单元分析模块，计算杆单元刚度矩阵
assembly.py      # 组装模块，生成对号矩阵和总体刚度矩阵
solver.py        # 求解模块，采用缩减法求解刚度方程
postprocess.py   # 后处理模块，计算应力、输出结果、绘制桁架图
truss_2_1.json   # 作业算例1：一维两单元杆结构
truss_2_2.json   # 作业算例2：二维两杆桁架结构

4.使用方法
编写如下的算例输入文件
{
  "Title": "算例标题",
  "nsd": 空间维度(1/2),
  "ndof": 每节点自由度(1/2),
  "nnp": 总节点数,
  "nel": 总单元数,
  "nen": 每单元节点数(恒为2),
  "CArea": [各单元横截面积],
  "E": [各单元弹性模量],
  "x": [各节点x坐标],
  "y": [各节点y坐标],
  "IEN": [[单元1节点1, 单元1节点2], [单元2节点1, 单元2节点2]],
  "fixed_dof": [约束自由度编号],
  "fixed_value": [对应约束位移值],
  "force_dof": [载荷自由度编号],
  "force_value": [对应载荷值],
  "plot_truss": "yes/no",  # 是否绘制桁架图
  "plot_node": "yes/no",   # 是否显示节点编号
  "plot_tex": "yes/no"     # 是否导出LaTeX绘图文件
}

5.注意事项
所有输入参数采用国际单位制（m, Pa, N），请确保单位统一
节点编号和自由度编号均从 1 开始，程序内部会自动转换为 0 索引
若出现中文字体警告，可将 JSON 文件中的Title改为英文，或在postprocess.py中设置中文字体
程序已兼容 Windows、Linux 和 macOS 系统
